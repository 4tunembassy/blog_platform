# Frontend Admin (Editor Console)

This folder will hold the Next.js app for the internal admin/editor console.

## Create (first time)
From `frontend/admin`:

```bash
pnpm dlx create-next-app@latest . --ts --eslint --app --tailwind --src-dir --import-alias "@/*"
```

Then run:
```bash
pnpm dev --port 3001
```
